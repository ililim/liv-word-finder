Mode dropdown — 6 resolutions

The current compact-mode menu nests fully-rounded pill rows inside a 16px-radius panel: two unrelated curvatures fighting in a 150px box. Each variant below resolves the clash a different way. Same trigger button everywhere (28px amber pill, PATTERN active) so only the menu changes. All dimensions at real compact scale: 40px rows, 2px borders, 850-weight uppercase.
